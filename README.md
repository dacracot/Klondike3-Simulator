# Solitaire simulator for finding the best strategy...

---
## How To:
- run `ant`
- run `java -jar simulator.jar`

## Versions
- 0.0 show a deck of cards
- 0.1 shuffle a deck of cards
- 0.2 create goal, stack
- 0.3 create board, play stack to goal

## play
- [ ] board to board
- [ ] board to goal
- [ ] goal to board
- [ ] stack to board
- [x] stack to goal
